# 2026-07-10 修正まとめ（発注書・基本契約書・業務仕様書・利用許諾条件書）

本日行った修正の一覧。テンプレ表示・フォーム・配信経路（DB/ディスク）ごとの注意点付き。

## 前提：テンプレの配信経路（重要）
- 本番 worker は `TEMPLATE_SOURCE=db`。DBに登録されたテンプレ（`document_templates` / `document_template_versions`）は **DBの更新SQL** が必要。
- `templates_config.json` は **2ファイル**（ルート＋`services/worker/templates_config.json`）。フォーム項目は **worker側**が配信するので、項目追加時は両方編集必須。
- ただし `individual_license_terms_v3` は **config非キー＝ディスク配信**（DB SQL不要・worker再デプロイのみ）。
