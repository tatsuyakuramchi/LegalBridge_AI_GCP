# 出版 個別利用許諾 ヒアリングシート（事業部記入用）

本シートは、**出版等利用許諾条件書**（必要に応じて**追加利用許諾条件書**）を作成するために、
事業部から条件をヒアリングするためのものです。記入後、法務部へご提出ください。

| 希望する契約締結日 | ____年__月__日（未定可） |

---

## 記入にあたっての注意

- **★ = 必須**項目です。空欄だと契約書を起票できません。
- 以下は**法務側で採番・補完**しますので記入不要です：
  - 条件書番号（`ARC-PUBT-YYYY-NNNN` を自動採番）
  - 被許諾者（＝株式会社アークライト）の住所・代表者名（自社マスタから補完）
- 選択項目は該当する `[ ]` に `x` を入れてください（例：`[x] 許諾する`）。
- 不明・未確定の項目は「**未定**」「**要相談**」と記載してください（空欄のままにしない）。
- 金額・料率は**税抜/税込の別**も明記してください。
